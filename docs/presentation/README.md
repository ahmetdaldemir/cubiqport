# CubiqPort — Sunum

Proje sunumu tek sayfa HTML olarak hazırlanmıştır; tarayıcıda açıp sunabilirsiniz.

## Kullanım

1. **Tarayıcıda aç:** `docs/presentation/index.html` dosyasını çift tıklayın veya bir HTTP sunucusu ile açın.
2. **Navigasyon:** Ok tuşları (← →) veya Space ile sonraki slayt; sayfa kaydırarak da geçiş yapılır.
3. **Sunum modu:** Tarayıcıyı tam ekran (F11) yaparak sunum modunda kullanabilirsiniz.
